Basic template

Use this basic template to get your app started.

<!DOCTYPE html>
<html lang="en">
  <head>
    <title>Your app title.</title>
    <meta http-equiv=Content-Type content="text/html;charset=utf-8" />
    <meta name="viewport" content="initial-scale=1, maximum-scale=1, user-scalable=no, width=device-width" />

    <meta name="apple-mobile-web-app-capable" content="yes" />
    <meta name="apple-mobile-web-app-status-bar-style" content="white" />

    <link href="mice-mobile.css" rel="stylesheet" />
  </head>
  <body>
    ...
  </body>
</html>

Tab bar

Use Ratchicons in the nav.tab to represent different sections of your app.

  <nav class="tab">
    <a class="active" href="#">
      <i class="icon home"></i>
      <span class="label">Home</span>
    </a>
    <a href="#">
      <i class="icon user"></i>
      <span class="label">Profile</span>
    </a>
    <a href="#">
      <i class="icon star"></i>
      <span class="label">Favorites</span>
    </a>
    <a href="#">
      <i class="icon search"></i>
      <span class="label">Search</span>
    </a>
    <a href="#">
      <i class="icon gear"></i>
      <span class="label">Settings</span>
    </a>
  </nav>

Tab bar (labels only)

If you don't want to use icons, that's okay too. The text will appear larger to use the additional space.

  <nav class="tab">
    <a class="active" href="#">Label</a>
    <a href="#">Label</a>
    <a href="#">Label</a>
    <a href="#">Label</a>
  </nav>

Typography

Use headings and paragraphs to title and describe sections of your app. Apply or wrap elements with .content to give the content space around the screen.

<div class="content">
  <h1>h1. Heading</h1>
  <h2>h2. Heading</h2>
  <h3>h3. Heading</h3>
  <h4>h4. Heading</h4>
  <h5>h5. Heading</h5>
  <h6>h6. Heading</h6>
  <p>Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod
  tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam,
  quis nostrud exercitation ullamco.</p>
</div>
